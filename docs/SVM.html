Support Vector Machine (SVM) Classification

Support Vector Machines (or SVMs for short) are a powerful supervised-learning model for text classification. A single SVM tries to find the best hyper-plane to split an n-dimensional dataset into two sections, such that each section contains only data-points of a particular class. If (as is often the case) no such hyper-plane exists, the SVM uses a kernel function to project each point in the dataset into an additional dimension, and tries again, repeating this procedure until a hyperplane exists which can cleanly divide the dataset.

As such, a single SVM is primarily used for binary classification, although $n-1$ SVMs can be used in concert with each-other to classify a dataset of $n$ classes.

The Procedure

For this project, I'll be using SVMs to classify the Ground News dataset along the Bias, Factuality, and Owner Type categories, as outlined in the Naive-Bayes tab above. I'm comparing the SVM model to Decision Trees and Naive-Bayes in it's ability to correctly classify the Ground News data. SVM is a gold standard in text classification, so I'm hoping that it can out-perform the other two models in this regard.

#Converting the Ground News Data into a WDM
def create_wdm(corpus,index,cv):
    df = pd.DataFrame(cv.fit_transform(corpus[index]).toarray(),columns=cv.get_feature_names_out())
    return df.filter(regex='(?u)[a-zA-Z]{3}+')
count_wdm = create_wdm(corpus,'title',MyVect)
tfidf_wdm = create_wdm(corpus,'title',MyVect_TFIDF)
wdms=[count_wdm,tfidf_wdm]
bias={'Far Left':-3,'Left':-2,'Lean Left':-1,'Center':0,'Lean Right':1,'Right':2,'Far Right':3}
factuality={'Low Factuality':0,'Mixed Factuality':1,'High Factuality':2}
owner_type={'Other':-1,'Corporation':0,'Government':1,'Independent':2,'Individual':3,'Media Conglomerate':4,'Private Equity':5}
for wdm in wdms:
    wdm['bias']=corpus['bias'].map(bias).fillna(0)
    wdm['factuality']=corpus['factuality'].map(factuality).fillna(-1)
    wdm['owner_type']=corpus['owner_type'].map(owner_type).fillna(1)

It's no better at classifying the data accurately, and it appears to have more far misses and less near misses than Naive-Bayes. Meanwhile, the owner_type appears (once again) to have no discernable correlation to text content.

Plotting Major Coefficients

One thing that you can do with SVM models is look at the model's coefficients, which represent features along which they are splitting the data. In the case of text data, these coefficients are tied to the frequency of certain terms within the data which the model uses to distinguish different classes. Here I graphed the 15 most useful terms for differentiating articles along the Bias category, according to the model.

In[74]:
def plot_coefficients(model=SVM_Model, colnames=TrainCount.columns, top_features=15,title='Bias'):
    ## Model if SVM MUST be SVC, RE: SVM_Model=LinearSVC(C=10)
    coef = np.absolute(model.coef_).sum(axis=0).ravel()
    top_coefficients = np.argsort(coef,axis=0)[-top_features:]
    # create plot
    plt.figure(figsize=(15, 5))
    colors = ["red" if c < 0 else "blue" for c in coef[top_coefficients]]
    plt.bar(  x=  np.arange(top_features), height=coef[top_coefficients], width=.5,  color=colors)
    feature_names = np.array(colnames)
    plt.xticks(np.arange(0, (top_features)), feature_names[top_coefficients], rotation=60, ha="right")
    plt.title('Linear SVM Most Significant Features: '+title)
    plt.show()
    

for i in range(3):

The top coefficients plot shows that the model is picking up on some significant features of the data, and reveals some really interesting words that are differentiating the different political slants.

It's no surprise to me that the word 'texas' is useful for differentiating right-leaning sources from left-leaning ones. I would bet money that 'jibe' is following the same trend. The presence of 'state' at number two could also be an indication that writers of a particular political slant are more likely to invoke issues of state's rights when righting about the transgender topic. I hadn't thought of the term woman being politicized, but in the transgender rights debate, who you refer to as a woman carries weight.

It's also nice to see 'criticise' making an appearance as one of the words differentiating high-factuality from low-factuality texts.

Overall, while this model wasn't the best at classification, the ability to look into its coefficients produces some incredibly useful insights into the nature politics withn the context of the transgender topic in the news.
