Clustering the Data¶

One important step in analyzing this large amount of text data is sorting it into groups. While some of my data comes pre-sorted into various labelled groups, much of the data (specifically the NewsAPI and WorldNewsAPI datasets) are un-grouped. For this data, I'll be using an unsupervised learning technique known as clustering to group the data into groups, or clusters.

K-Means Clustering¶

One of the most straightforward methods for clustering is known as K-means clustering. I'm going to try clustering the World-News API data into labelled groups by using k-means clustering on the text TFIDF 1-gram word-document matrix.

#It appears that the silhouette score is maximized round n=3, so we'll go with that one for now. 
# Now let's visualize the categories that have emerged.
km=KMeans(n_clusters=3,n_init='auto')
km.fit(wdm)
wdm['labels']=km.predict(wdm)
#Make a DF for the categorical term frequencies
wc_sums=pd.DataFrame(columns=wdm.columns)
#Each row of this df is a sum of the portion of the wdm which fits one category.
for l in range(3):
    wc_sums.loc[len(wc_sums.index)]=wdm[wdm['labels']==l].sum()
# Divide the dataframe by the mean along the y axis to get the relative frequencies of each term in each category
# compared to the average term frequency across the whole corpus.
wc_sums = wc_sums/wc_sums.mean()

wordclouds=[]
fig,ax=plt.subplots(3,1,constrained_layout=True)
for i in range(3):
    sums = wc_sums.iloc[i,:].sort_values(ascending=False)[:100]
    wordclouds += [WordCloud().generate_from_frequencies(sums)]
#Build the WordClouds
for i in range(3):
    ax[i].imshow(wordclouds[i])
    ax[i].set_title("Label "+str(i))

Patterns Emerging¶

K-Means has found three distinct clusters, each focused around a roughly distinct subject.

* Ron DeSantis and the Florida bathroom bill appears to be one cluster.
* The Brianna Ghey killing and possibly another British news story about a reality-TV celebrity named Ella Morgan Clark, though I'm not sure.
* The most clear-cut category. The Ohio Senate recently passed a bundle of laws relating to this issue, banning the use of 'gender-affirming care' on minors, with an emphasis on hormone blockers, as well as restricting athletes from competing in gender divisions other then the gender they were assigned at birth.

Let's run it another time to see if the same categories emerge.
